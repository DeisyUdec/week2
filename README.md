Ejercicio orientado en donde se implmenta uan red neuronal con la compurtas logica del AND, para llevar a cabo lo anterior se desarrollo de la sigueinte manera:
1. se importa la libreria de numpy para crear arreglos
2. se creo una funcion para retornar un valor 1 si se cumple la funcion si no es asi un valor 0
3. se crea una funcion denominada neurona en donde se determina la función de la entrada por el peso mas el sesgo.
4. se determinar lo valores
5. se determina las posibles entradas de las compuertas logicas del AND.
6. Se inicia el proceso de la red neuronal y muestra los resultados.

se crea una red neuronal basica de calcular la cantidad de porcentaje teniendo en cuenta el # personas para crear un diagrama de torta tiendo en cuenta quie la población final es de 60 personas, se logro implementado lo siguiente:
1. importar las librerias
2. ingresar los datos en entrada y salida
3. se crea la cantidad de capas de kera para ejecutar el modelo.
4. se crear el modelo de aprendizaje automatico tedienod en cuenta un ajuste en el peso y en el sesgo, ademas debe agregar la cantidad de taza de aprendizaje.
5. se ejecuta el modelo implementando las variables de entrada, salida y al cantidad de vueltas para que comienze el entrenaimiento.
6. se crea una grafica pra determinar la cantidad de epocas que se necesitan para ejeuctra el modelo
7. se pone a prueba el modelo
8. se muestra las variables del peso, sesgo que se necesito para dar a la solicitud

conclusión: Teniendo en cuenta el desarrollo del primer ejercicio referente la implemnetación de uso de las compuertas logicas del AND se obtuvo que cuando el valor de w1 es menor a 1 retorna un valor 0 por consigueinte todas las salidas de la compuerta logica dan 0 en cambio si colocamos un numero mayor 1 se cumple la funcion y se evidencia una salida 1 en la ultima compuerta lógica.
Además se puede evidenciar que con el desarrollo de una red neural siempre tiende a buscar una función genral determinando que cumpala con las condiciones para cada numero.
